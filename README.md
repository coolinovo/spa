# vue_zone

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### 项目开发
- 模式: 前后端分离
- 前端: 基于 Vue 技术栈的 SPA 项目
    - 技术选型:
        - Vue
        - vue-router
        - element-ui
        - axios
        - echarts
        - Aplayer
- 后端: 基于 node 开发的后端项目
    - node.js
    - express-generator
    - Jwt
    - MongoDB
    - mongoose
